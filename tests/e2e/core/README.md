# 核心流程与基础状态测试 (Core Flows)

此目录包含了双眼视觉康复系统中**非临床治疗**的核心基础功能测试。

## 测试范围
1. **基础信息流转**：
   - 测试首次进入系统时，收集年龄的 `UserInfoForm` 展现情况。
   - 验证废弃的“性别”等无关字段确实不存在。
2. **应用重置逻辑**：
   - 验证用户在 `HomeMenuLevel` 点击“清空数据并重置应用”后的行为。
   - 确保 `localStorage` 被清空且页面发生了**原位刷新**（`window.location.reload`），而不是路由的盲目跳转。

## 为什么将它们分开？
这些用例验证的是**系统级的基础底座功能**，无论后续如何增减“临床检查项目”或“康复训练阶段”，这套底座（进入前的档案收集、出错时的重置与清空）都必须绝对稳定。
