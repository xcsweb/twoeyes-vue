# 智能康复引擎测试 (Training Engine)

此目录包含了双眼视觉康复系统中最核心的**临床诊断与个性化康复联动**的逻辑测试。

## 测试范围
1. **年龄联动时长推算** (`personalized_training.spec.ts`)：
   - 验证系统是否根据不同年龄推断出神经可塑性（Neuroplasticity）差异。
   - 验证儿童（如10岁）的训练时长要求（10分钟）与成人（如25岁）的要求（20分钟）的精准区分。
2. **多态跳级解锁逻辑** (`intelligent_routing.spec.ts`)：
   - 验证统一的系统级流转函数（`useFlowManager().navigateForward`）与 `progressStore.effectiveUnlockedStage` 的联动机制。
   - **完全正常（无抑制、无斜视）**：全部阶段解锁，可直接点击所有阶段。
   - **仅斜视（无抑制、有偏移量）**：跳过前两阶段基础脱抑制，从阶段 3 开始。
   - **有弱视/单眼抑制**：锁定后续阶段，必须从头（阶段 1）开始逐阶训练。

## 为什么将它们分开？
这些用例验证的是**核心处方与康复引擎**（基于检查结果自动分配游戏与难度）。它们需要注入特定的 `localStorage` 数据来模拟眼科临床的各种复杂案例。
